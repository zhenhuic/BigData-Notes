# Hadoop 入门

## Hadoop 简介

1. 分布式存储系统 **HDFS** (Hadoop Distributed File System) **POXIS**
   - 分布式存储系统
   - 提供了高可靠性、搞扩展性和高吞吐率的数据存储服务
2. 分布式计算框架 **MapReduce**
   - 分布式计算框架（计算向数据移动）
   - 具有易于编程、高容错性和高扩展性等优点
3. 分布式资源管理框架 **YARN** (Yet Another Resource Management)
   - 负责集群资源的管理和调度
